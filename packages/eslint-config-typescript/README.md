# @abinnovision/eslint-config-typescript

> **Deprecated**: Merged
> into [@abinnovision/eslint-config-base](../eslint-config-base), which now
> includes TypeScript support by default.

## Migration

1. Replace the dependency:
   ```shell
   yarn remove @abinnovision/eslint-config-typescript
   yarn add --dev @abinnovision/eslint-config-base
   ```
2. Update your ESLint config to
   use `base`
   from `@abinnovision/eslint-config-base`. See
   the [eslint-config-base README](../eslint-config-base/README.md) for usage
   examples.
